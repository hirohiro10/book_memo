# 栞（しおり）— 読書メモ PWA

読書中に出会った良い一節を、栞をはさむように書き留めておくためのアプリです。
要約はせず、原文とひとことメモを残します。完全無料・端末内保存・オフライン対応。

## ファイル構成
- `index.html` … アプリ本体（これ1つで動きます）
- `manifest.webmanifest` … ホーム画面に追加するための設定
- `sw.js` … オフラインで起動するための仕組み
- `icons/` … アプリのアイコン

## 無料で公開する（家族に配る）手順
どれか1つでOK。いずれも無料です。

### A. GitHub Pages（おすすめ・URLが固定でずっと使える）
1. GitHub でリポジトリを作成
2. この4点（index.html / manifest.webmanifest / sw.js / iconsフォルダ）をアップロード
3. Settings → Pages → Branch を `main` / フォルダ `/ (root)` にして保存
4. 数分後に出る `https://ユーザー名.github.io/リポジトリ名/` を家族に共有

### B. Netlify Drop（最速・ドラッグ&ドロップ）
1. https://app.netlify.com/drop を開く
2. `shiori` フォルダごとドラッグ&ドロップ
3. 発行されたURLを共有

> ※ カメラ認識のコピペ整形やホーム追加は「https のサイト」で動きます。
> 上記A・Bはどちらも https なので問題ありません。

## スマホでの使い方
1. 共有したURLを開く
2. ブラウザの共有メニュー →「ホーム画面に追加」
3. アイコンから起動（以後は電波がなくても使えます）

## 基本の流れ
- **記録**：いま読んでいる本を選び、認識テキストを貼り付け → 自動整形 → 「栞をはさむ」
- **本棚**：本の追加（表紙撮影・章・タグ）、本ごとのメモ閲覧
- **見返す**：「今日の一節」ランダム表示・検索・タグで横断
- **設定**：データの書き出し／読み込み（機種変更・共有用）

## データについて
メモは各自のスマホの中だけに保存されます（サーバーに送られません）。
機種変更のときは「設定 → 書き出す」で保存したファイルを、新端末で「読み込む」。
家族でメモを見せ合いたいときも、この書き出しファイルを渡せば共有できます。
